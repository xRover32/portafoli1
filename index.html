Propiedad position

Ideas Principales

  • La propiedad position de CSS especifica cómo un elemento es posicionado en el documento
  • Las propiedades top, right, bottom, y left determinan la ubicación final de los elementos posicionados.
  • El position relative mueve ligeramente en base a su posición estatica
  • Con el position absolute los elementos se colocan en un contenedor padre
  • El position fixed es hermano del position absolute con algunas ligeras diferencias. Funciona exactamente igual, utilizando marcos de referencia, con la diferencia que el elemento se muestra en una posición fija
  • El position sticky se suele utilizar cuando queremos que un elemento se posicione en un lugar específico de forma fija

Apuntes Tomados

  • sticky

Resumen
